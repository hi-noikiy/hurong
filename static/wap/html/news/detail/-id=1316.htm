【小蚁】小蚁新闻双周刊第01期

2017-01-10

小蚁官方及负责人保证信息披露的内容真实、准确、完整，没有虚假记载、误导性陈述或重大遗漏。

为了让社区更加了解小蚁的项目动态，我们开通“小蚁新闻两周看”——每两周发布汇集有关小蚁技术开发、小蚁生态活动的资讯。（本次新闻会回顾以往一些重要的信息）

一、    技术更新

1、小蚁发布轻量级通用型虚拟机Antshares VM

11月14日，小蚁发布了虚拟机Antshares VM白皮书，并在GitHub上公布了开源代码。白皮书中描述了一种轻量级通用型区块链虚拟机Antshares VM，用户可以在小蚁区块链上部署并执行智能合约，让数字资产升级为可编程的智能数字资产。Antshares VM Github源代码地址：https://github.com/AntShares/AntShares.VMAntshares VM；白皮书地址：www.antshares.org/files/4007D7.pdf。

2、上线全新升级的轻钱包，用户体验更为友好

12月12日，小蚁官网上线了包括web端(chrome)和移动端（Android、iOS、Windows App）的轻钱包。此次升级，美化了钱包界面，从神秘紫换成了纯净蓝。与上一版本相比，还简化了钱包操作流程，增加联系人功能，让普通用户都易于使用。

现在版本的改进理念是突出钱包概念，只允许创建一个钱包，也只显示一个地址。因此和传统应用的账户类似，方便普通的互联网用户理解。

二、    战略合作

1、小蚁与Wings达成全方位战略合作，将共同开发市场和区块链技术

11月07日，小蚁和去中心化 DAO 平台 Wings宣布达成全方位战略合作。通过战略合作将使得海外的DAO项目能够进入中国的加密货币社区，而中国的创业项目也能到欧美社区中一展身手。

合作双方将对共同的商业机会进行合作，并通过共同参与的黑客马拉松活动来驱动整个生态的创新。此次合作还将促进区块链技术生态，为项目和企业在使用各自平台进行融资时建立更安全的标准。

2、小蚁和币创网达成战略合作，开拓资产数字化市场与区块链技术应用落地

11月17日，小蚁与国内数字资产新秀币创网达成战略合作，合作主要包括小蚁区块链上符合标准的优质资产可在币创网优先发行和流通，并将引入币创网上的优质数字化资产项目，此类项目通过小蚁区块链实现登记后，可通过小蚁生态及合作伙伴实现发行和流通。

其次，币创网CTO周邺飞将加入小蚁开发者社区，熟悉并研究小蚁代码。双方共同推动区块链技术的应用落地，对于合适的区块链项目，小蚁团队将与币创网联合进行业务拓展，并提供相应技术支持。

三、    社区活动

1、小蚁志愿者理事会成立，探讨社区运营工作计划和生态发展方向

为了更规范地组织和运营小蚁社区，实现志愿者团队的高效合作和发挥示范作用，11月24日小蚁正式成立了志愿者理事会。在成立会议上七位理事会成员亮相，介绍了工作计划。

2、小蚁发起创业者计划，开展技术和创业培训

小蚁发起基于小蚁生态的创业孵化活动。通过区块链创业项目征集、行业大咖评委点评、线上线下路演、投资者沙龙等形式，帮助创业Idea孵化、对接天使投资、技术合伙人等创业资源，并提供多样的退出渠道，帮助有创业想法的人加速区块链项目落地。

在12月01日，小蚁核心开发者林鹏涛通过直播分享区块链开发技术，其中内容有程序开发基础和C#版SDK结构。12月03日，小蚁理事会成员、抱诚守真信息科技创始人余晓菡邀请孵客创业公社投资总监路斌一同直播创业培训。

四、    生态资讯

1、反应链科技加入小蚁生态，微天使重装上阵

11月29日，成都反应链科技加入小蚁生态，此后作为小蚁生态前端的微天使（weangel.com）将由反应链科技专门负责运营。

微天使基于小蚁区块链以及小蚁生态，为各类资产项目提供登记、结算、发行、以及流通的一条龙配套便利服务。成都反应链科技主要定位于科技金融领域，其团队成员均为多年传统金融以及互联网金融的从业人员，对于私募基金投融资、一二级市场交易定价、线上平台运营有着丰富的经验。

2、微天使推出优质众筹项目，开拍学院币、小企股相继众筹成功

开拍学园和币创网入驻微天使平台，分别开启开拍学园币、小企股众筹。开拍学园是深圳开拍网科技有限公司旗下专注于艺人培训、影视从业培训与艺人经纪的子公司，开拍学园币（KAC）由开拍学园进行出售，出售所得所有收入用于开拍学园后续业务发展，于12月09日筹满份额。

继开拍学院币众筹在微天使平台首战告捷，币创网于12月15日发起众筹小企股应用生态链ICO，并在当天众筹到2000比特币。
