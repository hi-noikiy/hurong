【小蚁】小蚁新闻双周刊第03期

2017-01-13

小蚁官方及负责人保证信息披露的内容真实、准确、完整，没有虚假记载、误导性陈述或重大遗漏。

一、        技术更新

1.      PC钱包客户端进行了两次更新，分别增加了监视地址和兼容TESTNET网络的功能。监视地址功能允许你监控特定地址的余额和交易活动，但不能花费其中的资产，无须每次通过区块链浏览器查询，增强了公共账本的透明度；而客户端兼容TESTNET网络，通过修改配置文件切换主网和测试网，使开发者方便维护、同步升级。切换方法：http://8btc.com/article-4285-1.html。

2.      在开源系统Github发布了新的《节点部署》文档，介绍了在windows、server、 linux、docker等平台上记账节点以及私有链的部署优化，精简了记账节点的部署过程。文档地址：https://github.com/AntShares/AntShares/wiki/记账节点。

二、        社区活动

3.      小蚁团队将在下周举行年度主题直播活动，回顾2016年小蚁的发展历程，包括社区建设和技术进度等内容，总结这一年的经验得失，并交流2017年的项目展望。

4.        面向社区发布内容运营、渠道运营等兼职职位，主要工作内容为组织编辑小蚁教程以及在各大媒体、社区等渠道进行内容分发。

三、        市场合作

5.      经过初步沟通，某领域排名前列的网站，将尝试使用小蚁区块链为其客户进行资产发行的实验。

6.      为了市场需要将建立广泛的媒体合作，其中小蚁已在Xtecher注册推送资讯，这是一家优秀的科技创业媒体，致力于发现最有潜力的早期科技项目，报道科技领军人物和公司，并为它们进行全面的价值加速。

四、        生态资讯

7.      首家基于小蚁智能合约的超导模式交易所蓝鲸淘即将在春节前后落地，为小蚁链上各类数字资产提供流转渠道，为交易双方打造交易数据公开透明、不可篡改、可追溯查证、低成本的安全可信交易环境。
